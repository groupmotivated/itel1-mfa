# itel1-mfa
Motivated Finance App is an app that helps you manage your budget in a one app.



## Contributor guidelines
1. Fork the repo
2. Locally clone the repo
3. Open the folder
4. Open command prompt/terminal for that folder
5. Install all the dependencies  
   ``npm install``
6. Start the server
  ``npm run test``

7. Edit files 
8. Create branch to work with feature.
- WORK WITH ONLY ONE FEATURE!
- DO NOT CREATE FEATURES ON ``MAIN`` BRANCH.  
   This might render you getting diverged from the latest releases 
9. When starting work, always sync repo on your fork
